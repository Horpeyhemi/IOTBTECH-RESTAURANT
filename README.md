# IOTBTECH-RESTAURANT DATASET
### Overview ###
This dataset contains daily food records for a Nigerian Restaurant. Data contains orders from *Monday 4/20/2026* to  *Sunday 4/26/2026* across 4 locations in Lagos: **Lekki**, **Ikeja**, **Surulere**, and **Yaba** 
Use it for sales analysis, Pivot tables, Dashboards and practice Excel sheets. 

### Practice Questions ###
1. What is the Total revenue generated?
2. What is the meal with highest revenue?
3. which Location has the highest revenue?
4. What is the highest ordered meal?
5. What is the Top Location?
6. What is the Total Order Received?

### Key Details ###
1. Time Periods: 1 weeks, 15 Orders in Total
2. Categories: 2 Types - main Dishes drive the most revenue, plus starter and Drinks.
3. Currency: All prices are in Nigeria Currency(Naira)
